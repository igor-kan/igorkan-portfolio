# Personal Portfolio Website

This is a personal portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Single-page application** with smooth scrolling between sections.
- **Interactive hero section** with a Three.js particle system.
- **Detailed sections** for About, Skills, Research, Projects, Blog, and Contact.
- **Dark/light mode** with localStorage persistence.
- **Mobile-responsive design** for all breakpoints.
- **SEO optimized** with meta tags and structured data.
- **Accessible** with ARIA labels and keyboard navigation.
- **Performant** with code splitting and image optimization.
- **Analytics tracking** for page views and interactions.
- **Easter egg** (Konami code).

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, create a `.env.local` file in the root of the project and add your Formspree form ID:

```
FORMSPREE_FORM_ID=your_form_id
```

Finally, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
